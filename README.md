# 사용 기술
- HTML
- CSS
- Javascript

#실습하며 알게된 것
- 반응형 웹 페이지 설계 방식
  - 미디어 쿼리 사용
  - document.documentElement.clientWidth를 사용하여 브라우저 사이즈 계산하여 모바일 판단하기
  - scrollIntoView() 메서드
  - window resize 이벤트
- SVG 파일과 CSS 애니메이션을 이용하여 움직이는 화면 설계
